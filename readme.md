#### 安装教程

1. git clone
2. composer install 安装依赖
3. 命令行中使用php artisan key:generate 生成.env
4. 在.env中配置数据库等信息
5. php artisan make:migration 创建需要的表
6. php artisan db:seed 自动填充表数据
